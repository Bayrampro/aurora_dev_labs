Page {

### Задание 1. Поле ввода и кнопка

В первом задании был создан элемент `TextInput`, предназначенный для ввода текста пользователем, и кнопка `Button`. Поле ввода размещается по центру страницы, а кнопка располагается под ним.

```qml
TextInput {
    id: textInput
    anchors.centerin: parent
    text: "Text"
}

Button {
    anchors.horizontalCenter: parent.horizontalCenter
    anchors.top: textInput.bottom
    text: "Button"
    onClicked: {
        console.log(textInput.text)
        console.log("Вывод в консоль при нажатии")
    }
}
```

Для кнопки используется обработчик события `onClicked`. При нажатии на кнопку в консоль выводится текст, введенный в `TextInput`, а также дополнительное сообщение о срабатывании обработчика.

Результат выполнения первого задания: создан простой интерфейс ввода данных и реализована реакция приложения на нажатие кнопки.

### Задание 2. Размещение визуальных элементов

Во втором задании были использованы базовые визуальные компоненты QML: `Rectangle`, `Text` и `Image`. Для их вертикального размещения на странице применен контейнер `Column`.

```qml
Column {
    anchors.centerIn: page
    spacing: 50

    Rectangle {
        id: rect
        width: 300
        height: 100
        color: '#00a86b'
    }

    Text {
        id: text
        text: qsTr("Text")
    }

    Image {
        id: img
        source: "img/img.jpg"
        width: 400
        height: 400
    }
}
```

Элемент `Rectangle` используется для вывода цветного прямоугольника с заданной шириной, высотой и цветом. Элемент `Text` отображает строку на экране. Элемент `Image` предназначен для вывода изображения из файла, при этом для него заданы размеры `width` и `height`.

Результат выполнения второго задания: на странице были размещены графические и текстовые элементы, а также изучено позиционирование через контейнер `Column`.

### Задание 3. Пользовательский сигнал и обработка нажатий

В третьем задании была реализована работа с пользовательским сигналом. На странице создан контейнер `Column`, расположенный по центру экрана. В нем размещены четыре кнопки, каждая из которых вызывает общий сигнал и передает свой номер.

```qml
Column {
    id: root
    anchors.centerIn: parent
    spacing: 30

    signal buttonOnClicked(int buttonNumber)

    Button {
        text: "Button 1"
        onClicked: root.buttonOnClicked(1)
        backgroundColor: '#00a86b'
    }

    Button {
        text: "Button 2"
        onClicked: root.buttonOnClicked(2)
        backgroundColor: '#00a86b'
    }

    Button {
        text: "Button 3"
        onClicked: root.buttonOnClicked(3)
        backgroundColor: '#00a86b'
    }

    Button {
        text: "Button 4"
        onClicked: root.buttonOnClicked(4)
        backgroundColor: '#00a86b'
    }
}
```

Для контейнера объявлен пользовательский сигнал `buttonOnClicked`, который принимает целочисленный параметр `buttonNumber`:

```qml
signal buttonOnClicked(int buttonNumber)
```

При нажатии на кнопку выполняется обработчик `onClicked`, который вызывает сигнал `root.buttonOnClicked(...)` и передает номер выбранной кнопки.

Обработка сигнала выполнена через объект `Connections`:

```qml
Connections {
    target: root
    onButtonClicked: {
         console.log("Button", buttonNumber, "clicked!")
    }
}
```

Результат выполнения третьего задания: при нажатии на любую кнопку в консоль выводится сообщение с номером выбранной кнопки, например `Button 1 clicked!`.
