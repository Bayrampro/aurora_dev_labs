## Цель работы

Изучить базовые принципы создания мобильного приложения для Aurora OS с использованием Qt Quick, QML и Sailfish Silica: размещение элементов интерфейса, настройку свойств компонентов, обработку нажатий кнопок, создание пользовательских сигналов и подключение обработчиков событий.

## Задание

В рамках лабораторной работы необходимо было подготовить проект мобильного приложения и реализовать примеры работы с QML-компонентами:

1. Создать поле ввода `TextInput` и кнопку `Button`, при нажатии на которую текст из поля выводится в консоль.
2. Разместить на странице графические элементы: прямоугольник `Rectangle`, текстовый элемент `Text` и изображение `Image`.
3. Создать несколько кнопок, подключить к ним пользовательский сигнал и вывести в консоль номер нажатой кнопки.

## Ход выполнения работы

Проект создан как приложение Aurora OS. Точка входа находится в файле `src/main.cpp`. В нем создается объект приложения, задается имя организации и приложения, после чего загружается QML-интерфейс:

```cpp
view->setSource(Aurora::Application::pathTo(QStringLiteral("qml/Lab1.qml")));
```

Главный QML-файл `qml/Lab1.qml` описывает окно приложения `ApplicationWindow`, указывает начальную страницу `pages/MainPage.qml` и страницу обложки `cover/DefaultCoverPage.qml`.

Основная часть лабораторной работы реализована в файле `qml/pages/MainPage.qml`. Для страницы задан белый фон и разрешены все ориентации экрана:

```qml
Page {
    objectName: "mainPage"
    allowedOrientations: Orientation.All
    backgroundColor: '#ffffff'
}
```

В первой части работы был подготовлен пример с `TextInput` и `Button`. При нажатии на кнопку выполняется обработчик `onClicked`, который выводит содержимое поля ввода в консоль с помощью `console.log`.

Во второй части были рассмотрены базовые визуальные компоненты QML. Для размещения элементов использовался контейнер `Column`, внутри которого находились `Rectangle`, `Text` и `Image`.

В итоговой активной версии страницы реализовано третье задание. На странице создан контейнер `Column`, расположенный по центру экрана. В нем размещены четыре кнопки:

```qml
Column {
    id: root
    anchors.centerIn: parent
    spacing: 30

    signal buttonOnClicked(int buttonNumber)

    Button {
        text: "Button 1"
        onClicked: root.buttonOnClicked(1)
        backgroundColor: '#00a86b'
    }
}
```

Для контейнера объявлен пользовательский сигнал:

```qml
signal buttonOnClicked(int buttonNumber)
```

Каждая кнопка вызывает этот сигнал и передает свой номер. Обработка сигнала выполнена через объект `Connections`:

```qml
Connections {
    target: root
    onButtonClicked: {
         console.log("Button", buttonNumber, "clicked!")
    }
}
```

В результате при нажатии на любую кнопку в консоль выводится сообщение с номером выбранной кнопки.
